# Elsevier Submission Package

The `paper` directory contains the complete FISA 1.0 submission package:

- Elsevier CAS double-column source and local class dependencies;
- verified author, affiliation, email, ORCID, and CRediT metadata;
- manuscript sections, equations, algorithms, tables, and appendices;
- bibliography and publication figures;
- graphical abstract source and PDF;
- compiled manuscript PDF; and
- result traceability from manuscript claims to experiment artefacts.
