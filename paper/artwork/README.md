# Submission artwork

The manuscript contains four numbered figures.

| Figure | Submission file | Format |
|---|---|---|
| Figure 1 | Embedded TikZ text graphic in the LaTeX manuscript | Editable LaTeX |
| Figure 2 | `Figure_2_Range_Scaling.pdf` | Vector PDF |
| Figure 3 | `Figure_3_Gap_Coding_Comparison.pdf` | Vector PDF |
| Figure 4 | `Figure_4_Structural_Controls.pdf` | Vector PDF |

PNG alternatives are supplied at 300 dpi. The PDF files are the preferred
submission artwork because the plots are vector drawings. Colors are
supplemented by distinct markers, line styles, or hatch patterns so that the
series remain distinguishable without relying on color alone.
